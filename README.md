# Hackathon — WikiGenius

**Team:** ChickenRice
**Direction:** AI-Native Products & Operations
**Product:** WikiGenius — an AI-native team knowledge product with a two-tier agent system

## Folder Structure

| Folder | Purpose |
|--------|---------|
| `01_Proposal/` | Project proposal, problem statement, vision. Source of truth for what we're building and why. |
| `02_Research/` | Competitive landscape, user research, technical references, market signals. |
| `03_Design/` | Product spec, UX flows, system architecture, agent protocol design, data model. |
| `04_Build/` | Source code, prototypes, experiments, technical notes. |
| `05_Demo/` | Demo script, screen recordings, walkthrough materials, talking points. |
| `06_Submission/` | Final deliverables — submission form content, judging materials, slide deck. |
| `assets/` | Shared visual assets — logos, diagrams, screenshots, brand elements. |
| `pappers/` | Academic papers and literature references supporting the research phase. |
