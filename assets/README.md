# Assets

Shared visual assets used across proposal, deck, demo, and submission.

## Suggested contents

- `logo/` — WikiGenius logo (svg, png at multiple sizes)
- `diagrams/` — architecture diagrams, agent flow diagrams, permission model visuals
- `screenshots/` — UI captures, ideally with consistent framing
- `brand/` — colour palette, typography choices, any visual style notes

Keep filenames descriptive and stable — these get referenced from multiple docs.
