# 06 — Submission

Final deliverables for the hackathon submission and judging.

## Suggested contents

- `submission_form.md` — copy/paste content for the official submission form
- `slide_deck.pptx` — the deck used during judging (or link to it)
- `one_pager.pdf` — executive summary for judges who skim
- `final_demo_video.mp4` — the polished recording
- `team_info.md` — team members, roles, contacts

## Pre-submission checklist

- [ ] Submission form filled and submitted before the deadline
- [ ] Slide deck exported to PDF as a backup
- [ ] Demo video uploaded to a stable URL (not just a local file)
- [ ] All team members listed correctly
- [ ] Repo / prototype link works from an incognito window
